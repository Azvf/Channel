## 🎯 文档分类

### 稳定层（Principles）

阐述**"为什么我们要这么做"**，是项目的灵魂：

- **框架开发规范** - 分层架构、依赖规则、设计模式
- **交互与体验开发手册** - 乐观更新、物理感原则、渲染性能
- **无障碍设计规范** - 无障碍访问、微文案、错误处理

### 易变层（Reference）

**"怎么做"**的参考手册，每次 Commit 都可能变化：

- **设计系统规范** - 设计 Token 语义说明
- **工程化与构建指南** - 构建、检查、代码生成、环境配置

### 元文档（Meta）

文档编写和维护的标准：

- **文档开发与维护规范** - 文档编写标准
- **文档索引** - 完整文档清单

### Cursor Agent 文档

Agent 交互协议与开发规范：

- **Cursor Agent 交互简谱** - Agent 交互协议与 Prompt 规范
- **Cursor Agent 通用开发规范指南** - Agent 架构策略与开发规范
